## Descripción General

Este repositorio tiene como finalidad guardar las practicas de la materia extracción de datos en donde se aplican técnicas de extracción, limpieza, transformación y análisis de datos mediante el uso de herramientas de ciencia de datos. Durante el desarrollo se trabajó con diferentes conjuntos de datos para identificar inconsistencias, corregir errores y obtener información útil para la toma de decisiones.

## Objetivo

Desarrollar prácticas de análisis de datos utilizando Jupyter Notebook, aplicando procesos de limpieza, transformación y exploración de datos para generar información confiable a partir de diferentes fuentes de datos.

## Instrucciones de Ejecución

1. Clonar el repositorio:

```bash
git clone https://github.com/Brendis-max/ECBD-Brenda.git
```

2. Ingresar al proyecto:

```bash
cd ECBD-Brenda
```

3. Instalar las dependencias necesarias:

```bash
pip install pandas numpy matplotlib jupyter
```

4. Ejecutar Jupyter Notebook:

```bash
jupyter notebook
```
